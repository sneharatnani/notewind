# NoteWind

NoteWind is a note-taking app, created with ReactJS and Firebase.

## Key Features

- Sign In / Sign Out

- Toggle Layout / Change note background

- Sort / search / filter notes

- Pin / Unpin notes

- Add, update, and remove labels

- Fully responsive design

## Technologies Used
